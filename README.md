# Restaurant Menu & Order Management System - Part 2

In this assignment, I built a Restaurant Order Management System using Python data structures like dictionaries, lists, and nested dictionaries.

## Task 1 - Explore the Menu
In this task, I printed the restaurant menu grouped by category such as Starters, Mains, and Desserts. I displayed item name, price, and availability status. I also calculated total number of items, available items, most expensive item, and items priced under 150.

## Task 2 - Cart Operations
In this task, I created a cart system using a list of dictionaries. I implemented add item, remove item, and update quantity operations. I also checked item availability and handled invalid items. Finally, I generated an order summary with subtotal, GST, and total amount.

## Task 3 - Inventory Tracker with Deep Copy
In this task, I created a backup of inventory using deep copy. Then I updated stock based on the cart items and printed reorder alerts for low stock items.

## Task 4 - Daily Sales Log Analysis
In this task, I calculated total revenue per day and found the best selling day. I also found the most ordered item. Then I added a new day to the sales log and updated the revenue report. Finally, I printed a numbered list of all orders.
